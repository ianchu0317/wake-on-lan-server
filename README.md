# wake-on-lan-server
Server to manage wake on lan inside local network

### Features 
- Simple authentication
- Add Server (Name, IP or Hostname, MAC) to the list
- Edit Server information from list
- Delete Server from list
- Start / Power On server
- Stop / Shutdown server
- Deploy with docker (Backend, Frontend, DB)


## Project Structure

## Start server
```bash
docker compose up -d
```
